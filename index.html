Practicas Profecionalizantes I

HTML

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar páginas web. Permite definir títulos, párrafos, listas, enlaces, imágenes y más.

CSS

CSS (Cascading Style Sheets) se usa para dar estilo a documentos HTML. Controla colores, tipografías, tamaños, márgenes y diseño responsivo.

Git

Git es un sistema de control de versiones que permite guardar y gestionar los cambios en proyectos de programación, trabajar en equipo y mantener un historial.

GitHub

GitHub es una plataforma en línea que almacena repositorios Git. Facilita la colaboración, el control de versiones y el trabajo en proyectos de software.

MySQL

MySQL es un sistema de gestión de bases de datos relacional. Permite almacenar, organizar y consultar información de manera eficiente utilizando SQL.

Arquitectura de computadores

Materia 2

Computadoras y su funcionamientos

Una computadora es un dispositivo electrónico capaz de recibir, procesar y almacenar información. Su funcionamiento se basa en la interacción entre hardware y software.

Funcionamiento de computadoras

El funcionamiento de una computadora sigue el ciclo básico de: entrada, procesamiento, almacenamiento y salida de información. La unidad central de procesamiento (CPU) coordina estas operaciones.

Algoritmos a nivel bits

Los algoritmos a nivel de bits trabajan directamente con operaciones binarias, manipulando 0s y 1s para optimizar cálculos, cifrados y procesos de bajo nivel.

Periféricos

Los periféricos son dispositivos externos que permiten la interacción entre el usuario y la computadora. Se dividen en periféricos de entrada (teclado, mouse), de salida (monitor, impresora) y de almacenamiento.

Memorias

La memoria de una computadora almacena datos y programas. Se clasifica en primaria (RAM, caché) y secundaria (discos duros, SSD). Su velocidad y capacidad son fundamentales para el rendimiento.
